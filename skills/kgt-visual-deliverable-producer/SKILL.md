# KGT Visual Deliverable Producer

Create files when possible.

## Load Order

1. `01_STRATEGY/visual_identity_WORKING.md`
2. `08_MARKETING/marketing_context.md`
3. `10_ASSETS/` relevant assets via `kgt-asset-inventory`
4. task-specific context file

## Procedure

1. Create a KGT Visual Production Brief.
2. Use real KGT assets first.
3. Choose the right available tool/model.
4. Save output under `10_ASSETS/exports/YYYY-MM-DD_slug/`.
5. Save final output, source/editable file if possible, preview, notes, and QA report.
6. Run `kgt-quality-gate` before public use.

If a needed plugin/tool is missing, create the brief and notes, then name the missing tool/plugin exactly.

Output: export folder path, files created, assets used, QA result, limitations.
