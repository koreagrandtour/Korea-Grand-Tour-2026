# GitHub Safety Checklist

Use before every commit and push.

## Repository

- [ ] Repository is private.
- [ ] Remote URL points to the intended Korea Grand Tour GitHub account.
- [ ] `.gitignore` is present at the repository root.
- [ ] No push is made unless François explicitly asks.

## Private Data

- [ ] No participant phone numbers.
- [ ] No participant emails.
- [ ] No plate numbers.
- [ ] No insurance files.
- [ ] No driver license/passport files.
- [ ] No payment proof.
- [ ] No signed agreement PDFs.
- [ ] No private Kakao/private participant conversations.
- [ ] No confidential sponsor negotiation notes.

## Credentials

- [ ] No `.env`.
- [ ] No token JSON files.
- [ ] No API keys, private keys, PEM/P12/PFX files.

## Heavy / Sensitive Assets

- [ ] No raw video files.
- [ ] No raw audio/narration files unless intentionally approved.
- [ ] No huge exports.
- [ ] No unrevealed route files.
- [ ] Images/media passed normal privacy/content/route/file-size review. Visible number plates do not require special plate-specific review because Francois has approved plate display rights for images.

## Final Check

- [ ] Staged files are reviewed one by one.
- [ ] Public-facing docs separate confirmed facts from recommendations.
- [ ] 2025 actual metrics are not mixed with 2026 projections.
- [ ] Cross-check rules were applied for public/high-risk files.
