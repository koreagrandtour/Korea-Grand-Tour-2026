# Visual Identity - Working Guide

Last updated: 2026-05-27

This is a working guide, not a final brand book. Francois may update colors, fonts, and visual rules later.

## Brand Feeling

Korea Grand Tour should feel:

- warm
- human
- direct
- premium but not corporate
- energetic but safe
- selective but not arrogant
- adventurous and Korea-centered

The feeling is road rally, Korean landscape, car culture, friendship, culture/history, food, and memory.

## Working Colors

Current working colors:

- White
- Pungent blue
- Strong orange

Exact color codes are not confirmed yet. Do not present any hex code as official unless Francois confirms it.

## Fonts

Use the current website fonts as the working font direction.

OPEN QUESTION: Record exact font names from the website when the active website style is inspected.

## Logo / Name Rules

- Prefer `Korea Grand Tour` in public visual titles.
- Avoid `KGT` in public visual titles by default unless space or design constraints make it necessary.
- Public organizer should be `Korea Grand Tour Team`.

## Car Line-Up Visual Rules

Use only:

- car name
- `Korea Grand Tour 2026`

Do not include:

- `Registered Car`
- extra participant identity unless approved
- logo on car line-up visuals unless Francois changes the rule

People identity is not disclosed for public line-up posts for now.

## Metrics Rule

Use only verified 2025 metrics:

- approximately 2,000 reach
- approximately 4,000 views/video views
- approximately 50 followers gained
- 15 registrations

Do not use inflated or unverified numbers.

## Asset Rule

New Korea Grand Tour 2026 assets created for Instagram, YouTube, website, sponsors, or public communication should be saved in the relevant local folder and added to GitHub when public-safe, 2026-relevant, and reasonably sized.

Default destinations:

- `10_ASSETS/website_social_ready/`
- `10_ASSETS/exports/`
- `08_MARKETING/`
- `07_WEBSITE/`
- `06_SPONSORS_PARTNERS/`
